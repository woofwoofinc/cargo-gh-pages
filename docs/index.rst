Cargo Sphinx is an extension to the Rust_ language Cargo_ build tool which adds
features for building and publishing Sphinx_ documentation.

- Includes support for publishing to `GitHub Pages`_. See the example site at
  `woofwoofinc.github.io/cargo-sphinx`_.

.. _Rust: http://rust-lang.org
.. _Cargo: http://doc.crates.io
.. _Sphinx: http://www.sphinx-doc.org
.. _GitHub Pages: https://pages.github.com
.. _woofwoofinc.github.io/cargo-sphinx: https://woofwoofinc.github.io/cargo-sphinx
